용접로봇행동생성 데이터

RGB-D multi-view 및 형상 메시(mesh) 데이터를 결합한 멀티모달 Vision-Language-Action 모델 개발 — 산업용 용접 로봇의 행동 생성 데이터셋 유효성 검증을 목표로 한 응용 연구.

Overview

본 프로젝트는 한국지능정보사회진흥원(NIA) 주관 AI 학습 데이터 구축 사업의 일환으로, 용접 작업을 수행하는 로봇 매니퓰레이터의 행동 데이터를 RGB-D 다중 시점(multi-view) 영상과 함께 수집하고, 이를 Vision-Language-Action (VLA) 학습에 활용 가능한 형태로 검증·정제하는 연구입니다.

기존 robot manipulation 데이터셋이 주로 가정 환경 작업(파지·정렬 등)에 집중되어 있는 반면, 산업 환경에서의 정밀 작업(용접) 데이터는 상대적으로 희소합니다. 본 연구는 이러한 격차를 메우는 것을 목표로 합니다.

Approach

  • 다중 시점(multi-view) RGB-D 카메라 셋업을 통한 용접 작업 데이터 수집 파이프라인 설계
  • 3D 형상 메시(mesh) 데이터를 RGB-D 시점 정보와 정합(align)하는 캘리브레이션 절차 정립
  • Vision-Language-Action 모델 학습을 위한 데이터 라벨링 스키마 설계 (action primitive, language instruction)
  • 수집된 데이터의 학습 적합성 검증을 위한 베이스라인 VLA 모델 fine-tuning 실험
Role 본 프로젝트에서 캘리브레이션 파이프라인 설계와 멀티모달 데이터 정합 파트를 담당하였습니다. RGB-LiDAR 캘리브레이션의 사람 의존적 튜닝 문제를 해결하기 위한 자동화된 파이프라인을 함께 구축 중입니다.

Methods & Tech

RGB-D Multi-View 3D Mesh Alignment Vision-Language-Action Camera-LiDAR Calibration PyTorch ROS Python

Status

2025년 8월 착수, 현재 진행 중. 데이터 수집 파이프라인 1차 검증 완료, VLA 모델 베이스라인 실험 단계.
